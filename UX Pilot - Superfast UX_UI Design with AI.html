UX Pilot
Build a multi-page interactive prototype for a tourism website called "Iamiti – Taniti Island Tourism" in a low-fidelity wireframe style. Use a grayscale color palette, simple boxes for placeholders, and clear labels. Pages to include: 1. Home Page: - Header: LOGO placeholder, Nav links (Home, Dining, Lodging, Activities, Transportation, FAQ). - Hero: Large image placeholder, headline "Discover Iamiti", button "Explore Activities" (links to Activities). - Four quick-link boxes: Activities, Lodging, Dining, Transportation (all clickable to respective pages). - Footer placeholder. 2. Activities Page: - Header (same as Home). - Title: "Activities on Tahiti". - Two activity blocks: Image placeholder, short description placeholder, "Learn More" button (links to Activity Detail). - CTA button: "View Transportation Options" (links to Transportation). - Footer placeholder. 3. Placeholder Pages (Dining, Lodging, Transportation, FAQ, Activity Detail): - Each with a Title, Back/Home navigation, and minimal low-fi layout. Style & Interactions: - Maintain a clean, simple, low-fidelity grayscale aesthetic. - Ensure all navigation items, buttons, and quick-link boxes are interactive and link to the correct pages. - Every page must have a way to navigate back to Home.
